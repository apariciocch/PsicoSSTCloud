# PsicoSSTCloud

## Configuración rápida

1. Instala las dependencias:
   ```bash
   npm install
   ```
2. Copia el archivo de variables de entorno de ejemplo y actualiza los datos si fuera necesario:
   ```bash
   cp .env.example .env
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
